# Programação Estruturada em C++

Repositório com exercícios, exemplos e trabalhos desenvolvidos
na disciplina de Programação Estruturada do curso de Sistemas de Informação.

## Conteúdos abordados
- Lógica de programação
- Estruturas condicionais
- Laços de repetição
- Vetores e matrizes
- Funções
- Manipulação básica de arquivos

## Linguagem e ferramentas
- C++
- Code::Blocks
